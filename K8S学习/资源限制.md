# 对Pod的资源限制
```yaml
apiVersion: v1
kind: Pod
metadata:
  name: resource-demo
spec:
  containers:
  - name: demo-container
    image: nginx:1.14.2
    resources:
      requests:
        memory: "64Mi"
        cpu: "250m"
      limits:
        memory: "128Mi"
        cpu: "500m"
```
requests指至少，limits指至多。  

### 关于单位

- 内存单位可以是 Mi（mebibytes）、Gi（gibibytes）等，例如 64Mi = 64 * 1024 * 1024 字节。  
- CPU 单位使用 m（millicores）作为度量，1000m 等同于 1 个 CPU 核心
